FAQs
What is FrontEnd Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building.

Is Frontend mentor free?

Yes absolutely. It has some paid functions but its use is completely free

Can I use my own designs on Frontend Mentor?

Yes, you can use your own designs. However, the platform provides detailed design briefs and assets to help you focus on coding.

How can i get help if I'm stuck on a challenge?

You can visit the "solutions" section in Frontend Mentor where you can see other users' solutions and find inspiration.
